                  webpack #

                  webpack是一款强大的模块加载器兼打包工具，它能把各种资源，例如JS（含JSX）、coffee、样式（含less/sass）、图片等都作为模块来使用和处理。

                  1. webpack初体验 #

                  1.1 初始化git # 

                  $ mkdir webpack-demos && cd webpack-demos && git init

                  1.2 初始化项目 # 

                  $ npm init -y

                  1.3 增加.gitignore #

                  创建文件

                  $ touch.gitignore

                  在文件中增加以下内容

                  node_modules 
.idea

                  1.4 在项目根目录下创建src和build目录 #

                  src目录存放源码，build目录存放编译打包之后的资源

                  $ mkdir src build

                  1.5 增加以下文件 #

                  1.5.1 增加文件src/component.js # 

                  $ cd src && touch component.js

                  exports.name = 'zfpx';

                  1.5.2 增加文件 src/index.js # 

                  $ cd src && touch index.js

                  var comp = require('./component.js');
console.log(comp.name);

                  1.5.3 增加文件build/index.html # 

                  $ cd build && touch index.html

                  <script src="bundle.js"></script>

                  1.6 下载webpack和webpack-dev-server # 

                  $ npm install --save-dev webpack webpack-dev-server

                  1.7 创建webpack的配置文件 # 

                  $ touch webpack.config.js

                  配置webpack.config.js

                  var path = require('path');
module.exports = {
    entry: path.resolve(__dirname, 'src/index.js'),//入口文件
    output: {
        path: path.resolve(__dirname, 'build'),//输出路径
        filename: 'bundle.js' //输出文件名
    },
};

                  1.8 修改 package.json # 

                    "scripts": {
+    "build": "webpack"
  }

                  1.9 执行命令进行编译 # 

                  $ npm run build
